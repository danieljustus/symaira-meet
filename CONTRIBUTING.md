# Contributing

Thanks for contributing to `symmeet`.

1. Discuss meaningful changes in an issue before starting broad work.
2. Keep `SymMeetCore` independent of sibling Symaira repositories and capture
   frameworks.
3. Preserve local-first behavior and do not add telemetry or remote services.
4. Add focused tests and run `swift build`, `swift test`, and `make lint`.
5. Keep JSON contracts snake_case and do not write diagnostics to stdout in
   machine-readable modes.

By contributing, you agree that your work may be licensed under Apache-2.0.
